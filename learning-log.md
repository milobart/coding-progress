# Day 2 (05.01.2026)
Daily Commands (for 90% of all usage)
- git status # see what changed and what staged
- git add . # stage changes
- git commit
- git push # upload to GitHub repo
- git pull # download updates

Navigation & Setup
- git clone <url>
- git diff # see commit history
- git branch # list branches
- git checkout # create and switch branches
- git merge # merge work into main branch

Safety & Recovery
- git restore # unde uncommited changes
- git reset --soft HEAD~1 # undo last commit
- git stash # temporary save unfinished work
